## 개요

### 모듈 역할
Agent_1은 **뉴스 원문을 수집하여 번역, 요약, 키워드 추출을 수행하고, Risk Factor 관련성을 판정**하는 파이프라인의 첫 번째 모듈입니다.

### 파이프라인 위치
```
[뉴스 원문 입력] → [Agent_1: News Analyzer] → Agent_5 / Agent_2
```

### 주요 기능 (4개)
1. **번역** (`translate_to_korean`) - 영어 → 한글
2. **요약 생성** (`generate_summary`) - 반도체 공급망 관점 요약
3. **키워드 추출** (`extract_keywords`) - 공급망 리스크 관련 키워드 추출
4. **관련성 판정** (`filter_false_positive`) - Risk Factor 관련성 필터링

---

## 워크플로우

```
translate_to_korean
    ↓
generate_summary
    ↓
extract_keywords
    ↓
filter_false_positive
    ↓
[조건부 분기]
├─ is_relevant=True → Agent_5 / Agent_2로 전달
└─ is_relevant=False → 필터링 종료
```

## State 구조

### 입력 필드
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `news_id` | str | 뉴스 고유 ID |
| `title` | str | 원문 제목 |
| `summary` | str | 원문 요약 (선택) |
| `content` | str | 원문 본문 |

### 출력 필드
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `title_ko` | str | 한글 제목 |
| `summary_ko` | str | 한글 요약 (LLM 생성) |
| `content_ko` | str | 한글 본문 |
| `original_language` | str | "korean" \| "english" |
| `keywords` | List[Dict] | 추출된 키워드 배열 |
| `is_relevant` | bool | Risk Factor 관련성 여부 |
| `relevance_score` | float | 관련성 신뢰도 (0.0~1.0) |
| `relevance_reason` | str | 판정 근거 |
| `error` | Optional[str] | 에러 메시지 |

## 기능별 상세 설명

### 1. translate_to_korean (번역)

#### 📋 설명
영어 뉴스를 한글로 번역합니다. 한글 입력 시 자동 스킵합니다.

#### 🤖 사용 모델
- **모델**: `gpt-4o-mini`
- **온도**: 0.3
- **타임아웃**: 60초
- **응답 형식**: 텍스트 (구조화된 형식)

#### 📥 입력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `title` | str | 원문 제목 |
| `summary` | str | 원문 요약 |
| `content` | str | 원문 본문 |

#### 📤 출력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `title_ko` | str | 한글 제목 |
| `summary_ko` | str | 한글 요약 |
| `content_ko` | str | 한글 본문 |
| `original_language` | str | "korean" \| "english" |

#### 💬 프롬프트

**System**:
```
당신은 전문 번역가입니다. 기술/뉴스 콘텐츠를 정확하게 번역하세요.
```

**User**:
```
다음 뉴스를 한글로 번역해주세요.

**주의사항**:
- 기술 용어는 한글 표기 후 괄호에 영문 병기: "액화천연가스(LNG)"
- 화학식/약어는 원문 유지: NF3, SF6, ASML
- 고유명사(기업명, 지명)는 한글 음차 + 괄호 영문: "대만반도체제조회사(TSMC)"

**원문**:
Title: {title}
Summary: {summary}
Content: {content}

**번역 형식**:
Title: [번역된 제목]
Summary: [번역된 요약]
Content: [번역된 본문]
```

#### 📝 예시

**입력**:
```json
{
  "title": "China restricts rare earth exports to US",
  "summary": "Chinese government announced immediate halt...",
  "content": "The Chinese Ministry of Commerce stated..."
}
```

**출력**:
```json
{
  "title_ko": "중국, 희토류 대미 수출 제한",
  "summary_ko": "중국 정부가 즉각적인 수출 중단 발표...",
  "content_ko": "중국 상무부는 다음과 같이 밝혔다...",
  "original_language": "english"
}
```

### 2. generate_summary (요약 생성)

#### 📋 설명
한글 뉴스의 요약을 **반도체 공급망 관점**에서 3-5문장으로 재생성합니다.

#### 🤖 사용 모델
- **모델**: `gpt-4o-mini`
- **온도**: 0.3
- **타임아웃**: 30초
- **최대 길이**: 150자
- **응답 형식**: 텍스트

#### 📥 입력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `title_ko` | str | 한글 제목 |
| `content_ko` | str | 한글 본문 |
| `summary_ko` | str | 원본 요약 (선택, 참고용) |

#### 📤 출력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `summary_ko` | str | LLM 생성 요약 (최대 150자) |

#### 💬 프롬프트

**System**:
```
당신은 반도체 공급망 리스크 분석 전문가입니다.
```

**User**:
```
다음 뉴스를 반도체 공급망 관점에서 3-5문장으로 요약해주세요.

**제목**: {title_ko}
**본문**: {content_ko}

**요약 원칙**:
1. 객관적 사실 중심
2. 반도체 관련 정보 우선 (기업명, 소재명, 공급망 이슈)
3. 구체적 수치/날짜 포함 (있을 경우)
4. 최대 150자

**주의**: 반도체와 관련 없어 보이게 요약하지 말 것 (후속 단계에서 판정)
```

#### 📝 예시

**입력**:
```json
{
  "title_ko": "중국, 희토류 대미 수출 제한",
  "content_ko": "중국 상무부는 7월 1일부터 미국에 대한 희토류 수출을 전면 중단한다고 발표했다. 희토류는 반도체 제조의 핵심 원자재로..."
}
```

**출력**:
```json
{
  "summary_ko": "중국이 7월 1일부터 미국에 대한 희토류 수출을 전면 중단한다고 발표. 희토류는 반도체 제조의 핵심 원자재로 삼성전자 등 국내 기업에 간접 영향 예상. 미-중 무역 갈등 심화로 공급망 불안정 우려."
}
```

### 3. extract_keywords (키워드 추출)

#### 📋 설명
한글 뉴스에서 **공급망 리스크 관련 키워드**를 LLM으로 추출합니다.

#### 🤖 사용 모델
- **모델**: `gpt-4o-mini`
- **온도**: 0.2
- **타임아웃**: 30초
- **개수**: 최대 10개
- **응답 형식**: JSON

#### 📥 입력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `title_ko` | str | 한글 제목 (2배 가중치) |
| `summary_ko` | str | 한글 요약 |
| `content_ko` | str | 한글 본문 (3000자 제한) |

#### 📤 출력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `keywords` | List[Dict] | 키워드 배열 (최대 10개) |

**keywords 구조**:
```json
[
  {
    "keyword": "희토류",
    "score": 0.95,
    "category": "소재/부품/화학물질",
    "reason": "반도체 제조 핵심 원자재"
  }
]
```

#### 💬 프롬프트

**System**:
```
당신은 공급망 리스크 감지 전문가입니다.
```

**User**:
```
다음 뉴스에서 공급망 리스크 관련 키워드를 추출해주세요.

**제목**: {title_ko} (중요도 2배)
**요약**: {summary_ko}
**본문**: {content_ko}

**추출 카테고리 (6개)**:
1. [소재/부품/화학물질] NF3, 네온, 희토류, 실리콘 웨이퍼, 포토레지스트 등
2. [기업/기관] TSMC, 삼성전자, ASML, SK하이닉스, OO공장 등
3. [지정학/규제] 대중 수출통제, 반도체법, OFAC 제재, CFIUS 규제 등
4. [물류거점/경로] 호르무즈 해협, 수에즈 운하, 평택 공장, 대만 등
5. [리스크 이벤트] OO공장 화재, OO항만 파업, 지진, 수출 중단 등
6. [산업/제품 카테고리] 파운드리, 차량용 반도체, AI 반도체 등

**제외 기준**:
- 범용 경제 용어: "교통", "수출", "생산", "영향" (구체적 조치명 제외)
- 특정 명칭 없는 일반 운송수단
- 동사/형용사 파생 추상명사

**출력 형식 (JSON, 최대 10개)**:
{
  "keywords": [
    {
      "keyword": "희토류",
      "score": 0.95,
      "category": "소재/부품/화학물질",
      "reason": "반도체 제조 핵심 원자재"
    }
  ]
}
```

#### 📝 예시

**입력**:
```json
{
  "title_ko": "중국, 희토류 대미 수출 제한",
  "summary_ko": "중국이 7월 1일부터 미국에 대한 희토류 수출을 전면 중단...",
  "content_ko": "..."
}
```

**출력**:
```json
{
  "keywords": [
    {
      "keyword": "희토류",
      "score": 0.95,
      "category": "소재/부품/화학물질",
      "reason": "반도체 제조 핵심 원자재"
    },
    {
      "keyword": "중국",
      "score": 0.90,
      "category": "물류거점/경로",
      "reason": "희토류 주요 생산국"
    },
    {
      "keyword": "수출 제한",
      "score": 0.88,
      "category": "리스크 이벤트",
      "reason": "공급망 차질 유발 사건"
    }
  ]
}
```

### 4. filter_false_positive (관련성 판정)

#### 📋 설명
뉴스가 **8개 Risk Factor 중 하나 이상과 구체적으로 관련**이 있는지 LLM으로 판정합니다.

#### 🤖 사용 모델
- **모델**: `gpt-5.5` (복잡한 추론 필요: 8개 Risk Factor와의 구체적 관련성 판단)
- **온도**: 0.2
- **타임아웃**: 60초
- **임계값**: 0.5 (relevance_score >= 0.5이면 관련)
- **응답 형식**: JSON

#### 📥 입력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `title_ko` | str | 한글 제목 |
| `summary_ko` | str | 한글 요약 |

#### 📤 출력값
| 필드명 | 타입 | 설명 |
|--------|------|------|
| `is_relevant` | bool | Risk Factor 관련성 여부 |
| `relevance_score` | float | 관련성 점수 (0.0~1.0) |
| `relevance_reason` | str | 판정 근거 |

#### 8개 Risk Factor

| 대분류 | 세부 항목 예시 |
|--------|----------------|
| 1. 지정학 & 규제 | 수출입규제, 기업제재, 무역분쟁, 지역갈등 |
| 2. 공급집중 & 단일소싱 | 장비 독과점, 소재 단일 공급, 지역 집중 |
| 3. 원자재 & 희소물질 | 희토류, 특수가스(WF6, NF3), 원자재 가격 변동 |
| 4. 기술 & 지식재산 | 기술 탈취, 특허분쟁, 기술 단절, R&D 집중 |
| 5. 물류 & 인프라 | 항만 혼잡, 항공 운임, 물류 경로 차단, 전력 인프라 |
| 6. 사이버 & 데이터 | 공급사 사이버 공격, OT/ICS 보안, SW 취약점 |
| 7. ESG & Compliance | 분쟁광물, 탄소, 인권/노동, 환경 규제 |
| 8. 재무 & 신용 Risk | 협력사 재무 부실, 환율 변동성, 보험/신용 위축 |

#### 💬 프롬프트

**System**:
```
당신은 공급망 리스크 분석 전문가입니다.
```

**User**:
```
다음 뉴스가 반도체 공급망 Risk Factor와 관련이 있는지 판단해주세요.

**뉴스 제목**: {title_ko}
**뉴스 요약**: {summary_ko}

**8개 Risk Factor**:
1. 지정학 & 규제: 수출입규제, 기업제재, 무역분쟁, 지역갈등
2. 공급집중 & 단일소싱: 장비 독과점, 소재 단일 공급, 지역 집중
3. 원자재 & 희소물질: 희토류, 특수가스(WF6, NF3), 원자재 가격 변동
4. 기술 & 지식재산: 기술 탈취, 특허분쟁, 기술 단절
5. 물류 & 인프라: 항만 혼잡, 항공 운임, 물류 경로 차단
6. 사이버 & 데이터: 공급사 사이버 공격, OT/ICS 보안
7. ESG & Compliance: 분쟁광물, 탄소, 인권/노동, 환경 규제
8. 재무 & 신용 Risk: 협력사 재무 부실, 환율 변동성

**판정 기준**:
- TRUE: 8개 중 하나 이상과 **구체적으로** 관련
  - 예: "중국 희토류 수출 제한" → [원자재&희소물질 > 희토류 공급]
  - 예: "호르무즈 해협 봉쇄" → [물류&인프라 > 물류 경로 차단]
- FALSE: Risk Factor에 해당 없음
  - 예: "트럼프 주식 매입" → Risk Factor 해당 없음
  - 예: "AI 도입 인력 재고용" → 일반 고용 이슈

**출력 형식 (JSON)**:
{
  "is_relevant": true,
  "relevance_score": 0.95,
  "relevance_reason": "[원자재&희소물질 > 희토류 공급] 중국의 희토류 수출 제한은 반도체 제조 핵심 원자재 공급에 직접 영향"
}
```

#### 📝 예시

**입력 1 (관련 있음)**:
```json
{
  "title_ko": "중국, 희토류 대미 수출 전면 중단",
  "summary_ko": "중국 정부가 7월 1일부터 미국에 대한 희토류 수출을 전면 중단..."
}
```

**출력 1**:
```json
{
  "is_relevant": true,
  "relevance_score": 0.95,
  "relevance_reason": "[원자재&희소물질 > 희토류 공급] 중국의 희토류 수출 중단은 반도체 제조 핵심 원자재 공급에 직접 영향"
}
```

**입력 2 (관련 없음)**:
```json
{
  "title_ko": "트럼프 전 대통령, 자사 주식 추가 매입",
  "summary_ko": "도널드 트럼프가 자신의 회사 주식을 추가 매입했다고 밝혔다..."
}
```

**출력 2**:
```json
{
  "is_relevant": false,
  "relevance_score": 0.05,
  "relevance_reason": "개인 주식 매입 뉴스로 8개 Risk Factor와 무관함"
}
```
